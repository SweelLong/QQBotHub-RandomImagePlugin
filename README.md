## 随机图片插件的配置文件

-   "Key":                                                           // 获取图片的关键词
-	"AdminQQ":                                                 // 管理员的QQ号，主要的用途是在群内修改API
-	"AtOperator":                                                // 是【true】否【false】@提醒请求图片的人
-	"AdditionalWords"                                        // 附加词
-	"ImageAPI":                                                  // API

## 相关指令

- 【图】只需包含该文字即可触发随机图片
- 【api】只需包含该文字即可查看API的信息
- 【newapi】文字的开头为这个关键词才会触发即可修改api
